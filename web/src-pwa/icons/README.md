# PWA Icons

This directory contains icons for the PWA version of img2pic.

Place your app icons in the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- badge-72x72.png (for notification badge)

You can use the existing icons from public/icons/ as a starting point and scale them to the required sizes.

For best results, use a simple, clean design that works well at small sizes.
